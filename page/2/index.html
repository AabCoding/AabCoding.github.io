你想活出怎样的人生

发表于 2024-04-06 更新于 2024-04-07 分类于 句号

太典型的宫崎骏式电影了，奇幻的冒险+隐喻构成了电影的核心底色。

不过相比于《千与千寻》《天空之城》《幽灵公主》这种，本身叙事就很能撑住的，偏商业向的电影，《你想活出怎样的人生》则更加偏向于导演本人的表达，其叙事部分相对偏少，而隐喻则占据了相当多的篇幅。

而宫崎骏主要的表达则主要集中在两个方面：

1.对日本近代史的思考

一些相关的隐喻如下，不一定对，只是我个人的理解
外星陨石：黑船事件
火灾：东京大轰炸
塔：天皇制度
鹦鹉：纳粹与军国主义
投胎的幽灵：战争后出生的新一代日本人

那么从这个开始出发，电影的这一方面的表达也就比较好懂了，日本从黑船事件，收到了西方事件的巨大冲击，走入了混乱的政治生态，幕府倒台后各方势力想要去建立日本理想国，但是地上天国只能是一瞬的初心，人的业与恶导向了纳粹与军国主义的路线，那么结束一切的现在日本，又要走向什么样的未来呢？

2.对自己动画生涯的回顾

同样也有很多隐喻，我也把我的想法写下来
苍鹭：铃木敏夫
火美：宫崎美子
积木：宫崎骏之前导员的12部电影
红亭：应该来自宫崎骏之前的电影《红猪》
白色纸片：应该来自宫崎骏之前的电影《千与千寻》
投胎的幽灵：应该来自宫崎骏之前的电影《幽灵公主》

宫崎骏被铃木敏夫和高畑勋带入业界，而后活出了自己的人生：吉卜力。总结性质的回顾了所有创作过的电影，但是同时又担忧自己离开后摇摇欲坠的吉卜力。

懂了这两层的表达，那么相对来说就能够理解这部电影与其他商业化电影的区别。《你想活出怎样的人生》是借隐喻与童话来实现厚重表达与自我思考的作品，尽管在节奏安排上可能不太完美，前慢后快的编排相当程度的阻碍了观感的自然延续，但是我仍然认为这是一部还算值得一看的作品，打分的话，能给到7.5

Vini Bistro By Dicken

发表于 2024-04-06 分类于 Stay Alone

店名：Vini Bistro By Dicken
地址：广东省广州市海珠区红梅路方圆滨江大厦101铺
消费：649元（人均）
评分：7

和朋友找到的一家西餐厅，勉勉强强算是一顿fine dining，细致来写写。

餐前面包

一般，吃不吃都行

例汤

松露蘑菇汤：不错的汤，气泡很好的突出了松露的香气，是一个好的处理方式。

前菜

松露芝士薯条：还行？无功无过的一道菜，熟度，控油，调味都正常，但也是不怎么能找到惊艳感。

烟熏牛舌：烟熏风味与酸的结合很合适，也是一个好的处理方式，牛舌本身的滋味也挺不错。

主菜

澳洲冰鲜M5西冷（5成熟）：外焦里嫩，正确的熟度控制很大程度的把牛肉的风味给带了出来，好评的。

香煎银鳕鱼柳：同样无功无过的一道菜，香煎鳕鱼这种很难做得难吃的菜，只要是没有做出特色来，就很容易没有什么观感，调料和鳕鱼的香气接合得一般，鳕鱼本身的腥味也稍微过头了一点。

主食

综合海鲜烩饭：全餐唯一的不满点，海鲜味过浓，而又没有其他调料来综合调和，导致整道菜下来就只有这一个味道，丰富度严重不足。

环境

店名就叫Bistro（小酒馆/咖啡馆）那自然就是那种偏开放的空间，但是喝酒的氛围倒不是很有感觉，虽然我是不喝酒，但是我观察下来喝酒的人也不多。感觉是只有喧哗的糟粕而没有活跃的精华，略微差评。

服务

服务还算勤快，虽然不是大规格的一桌一服务员，但是至少也做到了及时响应，唯一美中不足的是牛排不换刀，感觉是有点怪异。

总的来说能给的7左右的评价，本来想给7.5，但是海鲜烩饭属实是不太行，扣0.5表示一下。

背包乱斗：福西法的宝藏

发表于 2024-03-24 更新于 2024-03-31 分类于 句号

锐评为《Super Auto Pet》和《Backpack Hero》的缝合怪，玩法兴趣来得快走得也快，并且感觉没学到SAP的精髓，SAP的更偏向系统性游戏，游戏制作人负责提供一个个有趣的机制，机制与机制之间的交互则由玩家自行探索创造，游戏很有策略深度，可玩性很强，并且耐玩，而背包乱斗感觉更像是非系统性游戏，游戏制作人负责提供一个交互表（也就是游戏内的合成表）玩家参照表来游玩，当然你可以说背包乱斗中依旧有很多的词条机制，依旧存在一些连锁，但是背包乱斗的数值设计导致了这部分的连锁收益相对较低，而合成的收益则占据了大头，这就导致了游戏的耐玩性严重不足，另外游戏的派生切换比较欠考虑，导致游戏相对来说非常的依赖运气，对正反馈的形成是有负面印象的。

当然，也正是因为缝合了一个好的模板，游戏的前期数据是不会差的，并且感觉游戏也在推，短期有好的收益也正常，但是个人判断仍然是：上线有限，长线不足。

评个分的话：7/10

《餐饮品牌塑造》

发表于 2024-03-17 更新于 2024-03-19 分类于 句号

一看标题，理论上应该奇怪，为什么我会看这种书？但事实上，我确实热衷于跨行业看书，并且我还自认为在这本书里学到了一些对我后面自己做项目的东西，以下是一些分享记录。

首先是讲了中国国内的餐饮行业现状，主要聚焦在三线城市及以上的，平均客单价在50-200的连锁餐饮领域。

然后讲了品牌的强大力量，带来的优势是什么，相关的考虑是什么。

最后分几个角度（定位，文案，视觉设计，菜品，审计）讲了讲怎么做餐饮品牌。

最后我对我自己的那个独游的项目又有了新的想法，玩法核心当然还是走游戏设计的那套方法论，但是在游戏包装，包括文案，美术设计等方面，有必要从品牌的角度来考虑，而不是单单把视角局限在一个游戏项目上，或者更准确的说是，什么包装更能具有一种独特的调性，这种调性能够去成为一系列游戏的共同设计语言，让玩家去形成管性。

在这个基础上我做了独游的初版美术风格设定，与一些方向上的定调。如果项目能做出来，再来结合项目分享。
